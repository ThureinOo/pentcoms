PentComs is an interactive cheat sheet, containing a curated list of offensive security tools and their respective commands for penetration testing.

Filter by **open ports**, **services**, **techniques**, **what you have**, and more to find exactly the command you need.

Inspired by [WADComs](https://wadcoms.github.io/), [GTFOBins](https://gtfobins.github.io/), and [LOLBAS](https://lolbas-project.github.io/).

Every command uses consistent placeholder values:
- **Target IP:** `10.10.10.1` | **Attacker IP:** `10.10.14.1`
- **Domain:** `test.local` | **Username:** `john` | **Password:** `password123`
